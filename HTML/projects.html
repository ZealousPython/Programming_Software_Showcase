Light Stream

Light Stream was a game created by a 4-man team to compete in the BPA Software Engineering Team competition; After the team submitted the game it was ranked in the top 20 nationaly for the competition. The game created using C# in the Unity Game Engine. The game itself is top-down action game where the player fights oncomming wave of enemies unique to each level. The player can play 3 different class related to the common roles seen in fantasy based settings.The picture above shows the final boss of the game a red dragon.

Galaga

In December 2021 a student creeated a rudementary version of the popular arcade game Galaga as part of an assignment to learn how to use C# in the Unity Game Engine. The game uses an imported nav-path preset that the enemies follow in order to make the player move around the screen. The enemies can also shoot back at the player and when the player runs out of lives the game ends and the score is displayed. Each of the games pixel sprites were created from scratch using the program Aesprite to create the assets.
